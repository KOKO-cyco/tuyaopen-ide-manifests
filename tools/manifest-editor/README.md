# TuyaOpen IDE Manifest Editor

Web-based editor for managing TuyaOpen IDE development boards and assets.

## Requirements

- Node.js 18+
- Git with authentication configured

## Getting Started

### 1. Fork & Clone

Fork the [tuyaopen-ide-manifests](https://github.com/tuya/tuyaopen-ide-manifests) repository, then:

```bash
git clone https://github.com/YOUR_USERNAME/tuyaopen-ide-manifests.git
cd tuyaopen-ide-manifests/tools/manifest-editor
```

### 2. Install & Start

```bash
npm install
npm start
```

Open `http://localhost:3000` in your browser.

### 3. Edit

- **Create board**: Click "+ Add New Board"
- **Edit board**: Click "✏️ Edit" on any board
- **Upload image**: Use "📤 Upload Image" in board editor
- **Delete board**: Click "🗑️ Delete" on board card

### 4. Commit & Push

Changes auto-commit with descriptive messages. After pushing to your fork, the editor guides you to create a pull request on GitHub.
