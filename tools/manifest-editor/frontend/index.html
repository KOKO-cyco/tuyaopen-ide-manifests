🛠️ Quick Start

1

Fork or Clone

Clone the manifest repository to your local machine or fork it on GitHub.

2

Create or Edit

Use this editor to create new boards or update existing ones. Changes are auto-committed with descriptive messages.

3

Validate

Schema validation ensures your data is correct. All URLs, formats, and references are checked automatically.

4

Push & Submit

Push changes to your branch and create a pull request. Our team reviews and merges high-quality contributions.

Fork Workflow:

  • Editor auto-detects your fork branch
  • After push, you'll see a "Create PR" guide
  • Click to open GitHub with pre-filled title & description
  • Review and click "Create pull request" on GitHub
  • Your PR is now submitted to the upstream repository!
