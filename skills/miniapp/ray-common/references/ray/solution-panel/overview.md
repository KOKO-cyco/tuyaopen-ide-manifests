面板小程序即面板开发方案，是专为智能设备控制面板设计的轻量化开发解决方案，基于小程序框架，支持跨平台开发和强大的 IoT 功能，帮助开发者快速、低成本地构建智能设备管理方案。

## 核心特点

- **简单易用**：采用 `JavaScript` / `Typescript` 开发，类似 `H5` 页面开发，上手快。
- **设备工具**：提供集成 `IDE` 和设备调试工具，支持从编码到测试的全流程开发，提升效率。
- **IoT 集成**：内置设备控制 `API` 和 `SDK`，简化 `IoT` 功能集成，实现智能管理。
- **多平台分发**：一次编写，适配 `Android` 和 `iOS`，节省开发时间。

### 唤起入口不同

### 内置设备环境

面板小程序运行时依赖虚拟或真实设备环境，内置设备 ID、群组 ID 和产品多语言支持。以下是获取设备 ID 和多语言的示例代码：

```typescript

// 获取设备 ID（单设备）

// 获取群组 ID（群组设备）
const { groupId } = getLaunchOptionsSync()?.query ?? {};

// 获取设备或群组对应的产品多语言值
```

## 方案流程

完整的开发流程请前往 [快速开始-面板小程序](/cn/miniapp/develop/ray/guide/start/quick-start) 进行查阅。

- **创建产品**：首先，我们需要在涂鸦开发者平台创建产品，面板开发工程师，硬件开发工程师以及产品设计师进行功能点的定义。

- **开发调试**：然后，面板开发工程师来开发围绕操作功能点的面板小程序，在开发阶段可以直接使用 Tuya MiniApp IDE 提供的虚拟设备进行调试。与此同时，硬件开发工程师也同样围绕功能点进行嵌入式程序的开发。

- **体验测试**：在联调阶段，面板小程序发布体验版本，通过智能生活 App 连接到真实设备进行测试（或者使用真机调试功能）。

- **审核上线**：测试完成后，我们可以将面板小程序提交审核，并发布正式版本。然后在 涂鸦开发者平台进行产品和面板小程序的绑定，那么，您就完成了面板小程序的整个开发工作了。

# 使用场景

面板小程序即面板开发方案为智能设备控制面板的开发提供了灵活、高效的解决方案，广泛适用于多种场景。

## 智能家居设备控制

面板小程序特别适合智能家居设备的控制面板开发，帮助用户通过移动端轻松管理家中的智能设备。

- **应用示例**：为智能灯具开发控制面板，支持亮度调节、色温切换和定时开关；为智能空调提供温度控制、模式选择和能耗监测功能。
- **用户价值**：用户可通过设备列表快速进入控制界面，实现远程操控，提升家居智能化体验。

## 工业物联网与设备管理

在工业场景中，面板小程序可用于设备状态监控和远程管理，提升生产效率和设备维护能力。

- **应用示例**：为工厂中的智能传感器或机械设备开发面板，实时展示运行状态（如温度、压力）、故障报警和历史数据。
- **用户价值**：企业管理者可随时查看设备运行情况，及时响应异常。

## 商业服务与集成

服务提供商和系统集成商可利用面板小程序为客户提供定制化的智能设备管理解决方案，尤其在中国市场与微信小程序集成更具优势。

- **应用示例**：为连锁酒店开发统一控制面板，管理房间内的灯光、窗帘和空调；为物业公司提供多设备管理界面，监控公共区域设施。
- **用户价值**：商业客户获得高效管理工具，提升服务质量和运营效率。

## 多设备联动与场景管理

面板小程序支持群组管理和场景联动，适用于需要同时控制多个设备的复杂场景。

- **应用示例**：为智能家居用户开发“一键回家”场景，联动打开灯光、空调和音响；为办公环境设计“会议模式”，统一调节灯光和投影设备。
- **用户价值**：用户通过单一界面实现多设备协同操作，简化使用流程。

## 开发者测试与快速原型

面板小程序为开发者提供高效的测试和原型设计平台，适合产品迭代和功能验证。

- **应用示例**：在开发新款智能插座时，快速构建控制面板原型，测试开关、定时和电量统计功能。
- **用户价值**：开发者可在虚拟设备环境中验证功能，缩短产品上市周期。

# 案例介绍

## 储能一体机

> 之前没有专门面向微型逆变器的设备控制面板，整体设计中式化，不太符合欧美人的审美观，且不具备接入防逆流电表的能力，影响到电表成品的出货。

针对欧美市场使用习惯重新设计的微型逆变器面板。首页展示光伏发电量及收益信息，用户轻松感知。面板感知设备在线状态，操作方便。此外该面板小程序接入了防逆流电表，行业内少数具备该功能，促进电表成品出货。

## 工业照明路灯

> 在全球“碳达峰”、“碳中和”的大背景下，商业照明应用场景采用智能化控制以减少碳排放，已经成为各国市政照明项目的首选解决方案。

采用面向商业照明应用的专业级 UI 面板设计，融合了“光感控制优先、人感三段式调光、恒照度感应自学习”的三合一传感功能，赋能客户快速落地市场订单，也为终端用户提供了绝佳的灯光体验与深度的节能效果。

## SMB 专业版照明

> SMB 方案定位是中高端专业照明应用市场，之前专业照明领域主要以有线系统 Dali、KNX 为主，因有线系统配置复杂专业程度高，我们的客户很难切入。

使用全新的产品设计交互理念，剥离日常使用功能和调试功能，让用户操作更清晰简洁，让施工调试更便捷专业。
