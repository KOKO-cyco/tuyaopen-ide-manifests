## 智能摄像机

本文主要介绍智能摄像机基于`涂鸦小程序`框架, 如何快速构建起自己专属的 **IPC 面板小程序**

### 功能介绍

智能摄像机品类基于小程序框架，支持多种类型摄像机的控制功能，包括：

- 视频类:  实时预览、视频回放（云端回放、SD 卡回放）、录制、对讲

- DP类: 云台&收藏点、巡航、隐私模式等

此外，还提供以下基于 AI 算法的增值服务：

- 智能消息

- 视频巡检

- 智能云存储等等

您可以通过下方[快速上手](/cn/miniapp/solution-panel/ability/special/player/ipc/quick-start/base)文档，快速体验 IPC 面板小程序的功能。

### 功能定义

摄像机产品的功能主要有以下分类:

**标准功能**: 平台内置的通用功能，支持基础属性的配置与调整。例如：红外夜视（枚举类型），您可以在支持的标准功能基础上扩展枚举值，实现自定义功能，并通过定制面板进行展示与扩展。

**自定义功能**: 您可根据业务需求，自定义新增功能点（DP 点），并通过面板进行交互控制，满足更灵活的产品功能设计

**高级功能**: 方案内置固定的增值功能，您需要勾选相关需要的功能，才可正常使用

产品功能绑定在产品方案上，基于产品方案，您可以通过 [IOT](https://platform.tuya.com/pmg/solution) 完成产品创建、功能绑定、面板绑定等操作。

### 面板选择

创建产品 => 01 功能定义 => 02 设备交互 => 面板控制 => 公版面板 => `IPC通用小程序面板`
