涂鸦目前在面板小程序业务中提供的物料和周边生态都是基于 `Ray` 开发的，因此强烈推荐使用 `Ray` 开发面板小程序。

`Ray` 是一款由涂鸦自研的跨端应用开发框架。它允许您使用 `React` 的语法进行开发，并将代码编译成智能小程序来运行。

## Ray 的优势

- **跨端开发** ：使用 React 语法，无需重新学习小程序语法，且不与特定业务绑定，可以用于开发任何形式和业务的小程序，此外也能够编译成微信小程序。
- **降低开发门槛** ：帮助现有涂鸦开发者（React Native）更方便地迁移到面板小程序开发。
- **生态兼容** ：复用 React 生态，能够更好的提高开发效率，同时能借助 [混合开发](/cn/miniapp/develop/ray/framework/mixed-development) 复用原生小程序的生态。
